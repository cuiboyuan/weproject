# App Name: WeProject

## How to start
- Install all the dependencies: `npm i`
- Build and run at the localhost PORT=3000: `npm start`
- Login Credential `username:password`
    - user: `user:user`
    - admin: `admin:admin`

## Third party libraries:
- React
- React Router Dom
- React uuid (creating unique id for new objects)
- react-dropzone (image upload)
- Ant Design (Avatar, Icons)
- React Bootstrap (SimpleCard, SimpleList)

## Current Features
- Header
    - Router
    - User logour
- Login
    - Sign In/Sign up
    - Login as admin
- Project Browsing
    - title
    - owner avatar, username
    - description
    - number of views/team membesr/likes
- Teammate Browsing
    - User info
    - Connect with users
- Project Details
- Teammate Details
